# pong

## Description

Pong implemented with JRuby and Slick2D.

![Demo](https://github.com/tobscher/pong/raw/master/demo.png)

## Prerequisites

* JRuby

## Start Pong

Run the following command to start a new game of pong:
```
jruby main.rb
```
